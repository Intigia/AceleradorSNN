# AceleradorSNN
El proyecto **AceleradorSNN** ("Sistema cognitivo neuromórfico para la detección de objetos en sector aeronáutico, automovilístico e industria"), desarrollado por **Intigia** y apoyado por fondos europeos NextGenerationEU, tiene como objetivo crear un sistema de percepción visual avanzado, rápido y de alta eficiencia energética,.

La gran innovación de este proyecto reside en la integración de dos núcleos tecnológicos implementados sobre hardware FPGA y ASIC:
*   **Un Procesador Neuromórfico (PNN):** Basado en Inteligencia Artificial de tercera generación mediante Redes Neuronales de Impulsos (SNN). Este módulo analiza los datos asíncronos provenientes de sensores de visión dinámica (cámaras DVS) para detectar objetos en la escena. Al procesar únicamente los cambios detectados (eventos), logra una velocidad asombrosa, latencia mínima y reduce drásticamente el consumo energético respecto a las redes neuronales convencionales.
*   **Un Procesador de Señal de Imagen (ISP) cognitivo:** Este componente trabaja en sinergia con el PNN, interpretando sus instrucciones para ajustar dinámicamente y en tiempo real los parámetros de una cámara RGB de alta resolución (como la exposición o el balance de blancos) ante los cambios de iluminación del entorno,.

Al fusionar la velocidad de las cámaras DVS con la calidad visual de los sensores RGB, el AceleradorSNN busca aumentar la precisión y la capacidad de adaptación en **vehículos autónomos (ADAS), drones aeroespaciales y robótica industrial**, optimizando además su autonomía en sistemas alimentados por baterías.
